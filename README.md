# Lightweight-Image-Watermarking-System-
Project Description:
Developed a lightweight image watermarking system to securely embed and extract ownership information from digital images. The system uses Digital Signal Processing (DSP) techniques for watermark embedding while leveraging OpenCV for image processing operations. Implemented in Python, the solution ensures minimal impact on image quality while providing efficient copyright protection and image authentication. The system supports watermark insertion, extraction, and verification with low computational overhead, making it suitable for real-time applications.

Key Features:

Invisible watermark embedding and extraction
Image ownership verification
High image quality preservation
Low computational complexity
Fast processing using OpenCV and DSP techniques
